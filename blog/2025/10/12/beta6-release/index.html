Player-Voiced Vanilla Dialogue (Dragonborn Voice Over Integration) @Dekanaâ€‹

SkyrimNet now supports voicing the player's vanilla dialogue lines with TTS. To enable this feature, download Dragonborn Voice Over but disable or delete its DBVO.esp file (no voice pack is needed). SkyrimNet will capture the dialogue event from DBVO and send it to TTS, rather than using a static voice pack. You can adjust the delay before an NPC responds to your dialogue selection in Advanced Configuration -> game -> Player-Voiced Vanilla Dialogue (DBVO). If the DBVO.esp file is active, then this feature will be disabled.
